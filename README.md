# DevShowcase — Portfolio Técnico

Portafolio web de desarrollo de software con proyectos fullstack, móviles, backend e infraestructura.

Incluye:
- Aplicaciones React
- Proyectos fullstack CRUD
- Apps móviles (Flutter / Kotlin)
- Sistemas backend PHP + MySQL
- Integración Docker + Open edX
- Proyectos de investigación tecnológica

## Stack del sitio

- React
- Vite
- Componentes reutilizables
- Layout responsive
- Despliegue en Vercel

## Proyectos destacados

- DemoWebStack CRUD — React + Node + Prisma
- PocketList — Flutter + Firebase
- FitBoost Trainer — Kotlin + Firebase
- Sistema CCTV — PHP + MySQL
- Open edX — Docker + Tutor

## Deploy

Sitio desplegado en Vercel.
